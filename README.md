## 目的
練習Nuxt3、Pinia狀態管理、串接API和typeScript。

## 啟動專案
```bash
git clone git@github.com:MikeLee0358/NuxtShoppingCart.git
cd NuxtShoppingCart
npm install
```

## 本地端運行
```bash
npm run dev
```

## 開啟JSON Server(供串接API)
```bash
json-server -p 4000 ./data/db.json
```
