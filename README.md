## 目的
練習Nuxt3和Pinia狀態管理、串接API。

## 安裝相關套件

```bash
npm install
```

## 開啟Nuxt伺服器
```bash
npm run dev
```

## 開啟JSON伺服器(練習串接API用)
```bash
json-server -p 3000 ./data/db.json
```
